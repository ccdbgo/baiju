# 白驹

白驹是一款一站式时间管理应用，当前仓库用于产品文档沉淀和 Flutter 客户端初始化。

## 目录

- `apps/baiju_app`：Flutter 主客户端
- `docs`：PRD、信息架构、技术选型和初始化方案
- `services`：后续服务端与后台任务预留
- `tooling`：后续脚本与开发工具预留

## 当前已完成

- 产品文档与技术选型文档
- Flutter SDK 安装到 `d:\code\tools\flutter`
- `apps/baiju_app` 脚手架初始化
- 基础路由壳层、主题、环境配置
- `Drift` 数据库骨架和首批关键表

## Windows 运行前提

如果要构建或运行 Windows 桌面端，需要先开启 Windows 开发者模式，避免 Flutter 插件 symlink 失败。
